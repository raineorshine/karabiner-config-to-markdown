- Back Tick: Option + Quote
  - L-Option + `'` → `` ` ``
- Better Braces: Alt + o/i
  - L-Option + `l` → `[`
  - L-Option + `;` → `]`
  - L-Command + L-Option + `l` → L-Command + `[`
  - L-Command + L-Option + `;` → L-Command + `]`
- Desktop Navigation: Right shift + brackets
  - R-Shift + `[` → L-Ctrl + `→`
  - R-Shift + `]` → L-Ctrl + `←`
